# 🍽️ FronteraPOS - Frontend

Frontend desarrollado en Vue.js para el sistema de gestión de restaurantes **FronteraPOS**.

## 📌 Descripción

Aplicación web desarrollada con Vue 3 que consume la API REST del backend de FronteraPOS.

Permite administrar:

- 📂 Categorías
- 🍽️ Productos
- 🪑 Mesas
- 📊 Dashboard

---

## 🚀 Tecnologías utilizadas

- Vue 3
- Vue Router
- Axios
- SweetAlert2
- Font Awesome

---

## 📷 Funcionalidades

- Inicio de sesión
- Dashboard dinámico
- CRUD de Categorías
- CRUD de Productos
- CRUD de Mesas
- Navegación protegida

---

## ⚙️ Instalación

Clonar el repositorio:

```bash
git clone https://github.com/zypher-sentinel/fronterapos-frontend.git
```

Instalar dependencias:

```bash
npm install
```

Ejecutar:

```bash
npm run serve
```
